# codes-fonciers-cen
Liste des codes sql, permettant la gestion et la visualisation des données foncières du CEN Pays de la Loire 

### codes_serveur_distant 
Code sql permettant d'envoyer les données foncières, du CEN Pays de la Loire, à la fédération des CEN de France.

### codes_bilan_foncier
Code sql permettant de récupérer les données foncières requises pour le bilan de fin d'année du CEN Pays de la Loire.
